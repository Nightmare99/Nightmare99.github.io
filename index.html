Projects

Below are a few projects I am proud of developing, though some are still under active development.

UnDanger

UnDanger is a women's safety device made using the Raspberry Pi microcontroller. It detects hotwords set by the user in real-time and alerts the authorities as well as emergency contacts, which are set by the user during setup of the device. I was mainly responsible for creating app for the device setup as well as the entire client-server architecture required for setting up the users and the hotwords for each user.

NotMoodle

A Google classroom-esque online learning platform. Anyone can create courses as well as join them. It is powered by ExpressJS under the hood and Materialize-CSS was used for the UI. This was created for and during the PayPal Value Added Program conducted in VIT Chennai.

Analysis of Lok Sabha elections

A Django website which presents the analysis of the Indian Lok Sabha elections over the years. Also uses HDFS (Hadoop File System) to store and retrieve the datasets. This project was done by me and 3 of my batchmates for a college course.

BattleShip

The original Hasbro Battleship board game, now online. There are 2 parts of this app - the client and the game coordinator, which is responsible for matchmaking between players. Both are written using JavaScript (NodeJS). The client also has an alternative platform as a discord bot (made using DiscordJS), but that is still under construction.

CoronaBot

A DiscordJS bot which sends daily corona statistics of Tamil Nadu, India, to a Discord server upon calling it with a special identifier. Inspired by one of my friends who shares the same corona statistics daily as a WhatsApp status.
